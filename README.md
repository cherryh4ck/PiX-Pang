### FAQs

**1. ¿Cómo puedo jugarlo fluidamente?**

> Vas a tener que jugarlo a través de Turbowarp, el cual es un fork de Scratch que corre de forma mucho más fluida. Si bien el juego está limitado a 30fps, Turbowarp permite usar la opción de 'Interpolación' (puesta por defecto) para que el juego se vea más fluido. Tené en cuenta que puede llegar a causar leves glitches visuales.

**2. ¿Cómo puedo jugarlo en una mayor resolución?**

> Esta recreación está hecha para solamente funcionar en 480x360 (res 4:3), pero usando el programa [Turbowarp Packager](https://packager.turbowarp.org/) podés exportar el proyecto original (.sb3) a una aplicación de Electron que tenga la opción de 'Estirar resolución' activada.

**3. Encontré un bug, ¿Cómo puedo reportarlo?**

> En el momento que encontrés un bug, presioná la combinación de teclas D + B + G, luego presioná la tecla Y y saca una captura. Creá un nuevo issue, adjuntá la captura y describí detalladamente el fallo.
